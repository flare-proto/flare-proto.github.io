    Setup Problems

    Partitioning

    what I did wrong: incorrect partitioning

    my setup: 

    mount point: /boot/efi
    format:      EFI system partition
    size:        100MB

mount point:
    format: btrfs
    size: [max]

mount point: /
    format:      btrfs (subvolume)

    Correct Partitioning

    mount point: /boot/efi
    format:      EFI system partition
    size:        300MB

mount point: /boot
    format:      ext4
    size:        1GB

mount point:
    format: btrfs
    size: [max]

mount point: /
    format:      btrfs (subvolume)

mount point: /var
    format:      btrfs (subvolume)

mount point: /var/home
    format:      btrfs (subvolume)
